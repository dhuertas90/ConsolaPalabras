	luego seleccionar la operacion deseada.

----------------------------------------------------------------------------------------------------------------------------------------------

INSTRUCCIONES DE USO:

- Se ejecuta 'Main.py', se vivualiza Menu principal.
- Se debe seleccionar una opcion (del 0 al 8) para realizar las siguientes operaciones respectivas:

Opcion 0: 
    DEVOLVER UNA PALABRA
    Para devolver una palabra se debe escribir una categoría existente, y se visualizará aleatoreamente una palabra de la respectiva categoría.
    O bien para regresar al Menu principal se debe escribir : atras.

Opcion 1:
    AGREGAR PALABRAS 
    Para agregar palabras se debe escribir una categoria existente y luego se debe escribir una palabra no existente.    
    O bien para regresar al Menu principal se debe escribir : atras.

Opcion 2:
    ELIMINAR PALABRAS
    Para eliminar una palabra se debe escribir una categoria existente y luego se debe ingresar la palabra a eliminar.
    O bien para regresar al Menu principal se debe escribir : atras.

Opcion 3:
    MODIFICAR PALABRAS
    Para modificar una palabra se debe escribir una categoria existente, luego se debe ingresar la palabra a modificar,
    y finalmente ingresar la nueva palabra.
    O bien para regresar al Menu principal se debe escribir : atras.

Opcion 4:
    MOSTRAR CATEGORIAS
    Para mostrar categorias (contenido de una categoria) se debe escribir una categoria existente.
    O bien para regresar al Menu principal se debe escribir : atras.

Opcion 5:
    AGREGAR CATEGORIAS 
    Para agregar una categoria se debe escribir una categoria no existente.
    O bien para regresar al Menu principal se debe escribir : atras.

Opcion 6:
    ELIMINAR CATEGORIAS
    Para eliminar una categoria se debe ingresar la categoria a eliminar.
    O bien para regresar al Menu principal se debe escribir : atras.

Opcion 7:
    MODIFICAR CATEGORIAS
    Para modificar una categoria se debe escribir una categoria existente y luego ingresar una nueva categoria.
    O bien para regresar al Menu principal se debe escribir : atras.   

OPCION 8:
    SALIR
